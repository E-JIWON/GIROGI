## 추가 비교 콘텐츠 아이디어 (10가지)

### 1. 집밥 비율 대결 (난이도: 쉬움)

**컨셉**: 이번 주 집에서 먹은 비율 비교
```
나: 🏠🏠🏠🏠🏠🏠🍽️🍽️  (75% 집밥)
친구: 🏠🏠🏠🍽️🍽️🍽️🍽️🍽️  (37% 집밥)
```

**왜 자극적인가**:
- 집밥 비율은 다이어트 성공과 직결 (연구: 집밥이 외식보다 평균 200kcal 적음)
- "나는 배달만 시키는데 쟤는 맨날 해먹네" → 행동 변화 유도
- 구현: 기존 MealPlace 데이터 활용 가능 (추가 데이터 불필요)

**심리학 근거**: Social Comparison Theory (Leon Festinger, 1954)

---

### 2. 미션 성공률 비교 (난이도: 쉬움)

**컨셉**: 주간 미션 달성률 비교
```
나:  ████████░░  80% (4/5일 성공)
친구: ██████████  100% (5/5일 성공)
      "친구가 이번 주 퍼펙트!"
```

**왜 자극적인가**:
- 100% 달성한 친구를 보면 "나도 할 수 있는데!" 심리
- 3개 중 2개 규칙이므로 100%가 어렵지 않음 → 따라잡기 가능한 목표
- 구현: DailyRecord의 coreMissions 데이터 활용

---

### 3. 유혹 저항 횟수 비교 (난이도: 보통)

**컨셉**: 총 유혹 저항 횟수 비교
```
나: 12번 참았어요!  🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️
친구: 18번 참았어요! 🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️🛡️
```

**왜 자극적인가**:
- "유혹을 참는 것"이 곧 강인한 의지 → 자존감 자극
- 친구가 더 많이 참으면 "나도 참을 수 있다" 동기부여
- User.totalTemptationResisted 필드 이미 존재

---

### 4. 건강 식재료 다양성 (난이도: 보통)

**컨셉**: 이번 달 먹은 건강 식재료 종류 비교 (뱃지 기반)
```
나: 단백질 8종 / 채소 5종 / 곡물 3종 = 총 16종
친구: 단백질 6종 / 채소 9종 / 곡물 5종 = 총 20종
      "친구가 채소를 더 다양하게 먹고 있어요!"
```

**왜 자극적인가**:
- "양"이 아닌 "다양성"으로 비교 → 체중 비교보다 건강한 경쟁
- "나는 계란만 먹는데 쟤는 연어도 먹네" → 식단 개선 유도
- 구현: Badge 카테고리별 획득 종류 수 활용

---

### 5. 주간 리듬 비교 - 요일별 기록 패턴 (난이도: 보통)

**컨셉**: 어떤 요일에 잘 하고 못 하는지 비교
```
       월  화  수  목  금  토  일
나:    ✅ ✅ ✅ ✅ ❌ ❌ ✅
친구:  ✅ ✅ ✅ ✅ ✅ ✅ ❌
       "당신은 금/토에 약해요! 친구는 주말을 거의 지켜요"
```

**왜 자극적인가**:
- 자신의 약점 요일을 인식 → 구체적 행동 계획 수립 가능
- "쟤는 금요일도 성공하는데" → 경쟁심 자극
- Implementation Intention 이론과 시너지 ("금요일 저녁에 집밥 해먹기")

---

### 6. 연속 기록 타임라인 (난이도: 보통)

**컨셉**: 누가 먼저 시작했고 누가 더 꾸준한지 시각화
```
1월       2월       3월
나:   ░░████░░████████████
친구: ░░░░██████░░░░████████
      "당신이 2주 먼저 시작했지만, 친구가 더 꾸준해요"
```

**왜 자극적인가**:
- "나는 오래 했는데 쟤가 나를 따라잡았다" → 위기감
- 끊긴 구간이 보이면 "다시 이어가자" 동기부여
- 장기 지속성 비교가 가능

---

### 7. "이번 주 MVP" 뱃지 (난이도: 쉬움)

**컨셉**: 매주 친구들 중 가장 잘한 사람에게 MVP 뱃지
```
🏆 이번 주 MVP: 운동왕
   - Streak 7일 유지
   - 미션 100% 달성
   - 유혹 3번 저항

   너는 2위야! 다음 주엔 네가 MVP!
```

**왜 자극적인가**:
- 듀오링고의 "리그 시스템" 효과 (상위 10%는 승급, 하위 10%는 강등)
- 매주 리셋되므로 누구나 1위 가능 → 희망 유지
- Social Facilitation 효과: 타인의 존재가 성과 향상

**심리학 근거**: Social Facilitation (Norman Triplett, 1898)

---

### 8. 보상 쿠폰 사용 비교 (난이도: 쉬움)

**컨셉**: 누가 더 많은 보상을 받았는지 비교
```
나: 🎟️🎟️🎟️ (치팅데이 3회 획득)
친구: 🎟️🎟️🎟️🎟️🎟️ (치팅데이 5회 획득)
      "친구가 더 꾸준해서 보상도 더 많아요!"
```

**왜 자극적인가**:
- "보상"은 노력의 증거 → "쟤는 저만큼이나 받았네" → 질투심 자극
- 쿠폰은 Streak 기반이므로 꾸준함의 지표
- 구현: CouponStore 데이터 활용

---

### 9. 시간대별 식사 패턴 비교 (난이도: 보통)

**컨셉**: 아침/점심/저녁 기록 비율 비교
```
        아침    점심    저녁
나:     30%    80%    60%
친구:   90%    85%    70%
        "친구는 아침을 거의 빠지지 않아요!"
```

**왜 자극적인가**:
- 아침 식사는 다이어트에 중요 (대사율 15% 증가)
- "나는 아침 안 먹는데 쟤는 맨날 먹네" → 구체적 행동 변화
- 구현: MealRecord의 mealTime 데이터 활용

---

### 10. 감량 속도 레이스 (난이도: 어려움)

**컨셉**: 목표 체중까지 남은 거리 비교 (게임화)
```
출발 ═══════════════════════ 목표
나:   ████████████░░░░░░░░░ 60%  🏃
친구: ██████████████░░░░░░░ 70%  🏃💨
      "친구가 10% 앞서 있어요! 따라잡으세요!"
```

**왜 자극적인가**:
- 레이스 형태의 시각화 → 게임처럼 느껴짐
- 목표 달성률이 보이면 "거의 다 왔다" 동기부여
- 친구가 앞서면 "조금만 더 하면 따라잡을 수 있다" 심리

## 핵심 원칙

### 건강한 경쟁을 유도하는 3가지 규칙

1. **절대값이 아닌 변화율로 비교**
   - ❌ "친구 체중: 65kg, 나: 80kg" (기분 나쁨)
   - ✅ "친구 감량률: 3%, 나: 5%" (노력 비교)

2. **항상 따라잡을 수 있다는 희망 제공**
   - 뒤처졌을 때: "3일만 더 하면 따라잡아요!"
   - 앞서있을 때: "계속 이대로!"

3. **실패를 비난하지 않고 격려**
   - ❌ "당신이 졌습니다"
   - ✅ "친구가 조금 앞서 있어요! 오늘부터 다시 시작해볼까요?"

### 심리학 근거

- **Social Comparison Theory** (Festinger, 1954): 비슷한 수준의 타인과 비교할 때 동기부여 극대화
- **Social Facilitation** (Triplett, 1898): 타인의 존재만으로 성과 향상
- **Goal Setting Theory** (Locke & Latham, 2002): 구체적이고 도전적인 목표가 성과 향상
- **Self-Determination Theory** (Deci & Ryan, 1985): 관계성(Relatedness) 욕구 충족 → 내재적 동기 강화
