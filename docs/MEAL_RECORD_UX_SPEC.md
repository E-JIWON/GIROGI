## 핵심 기능

### 1. 듀오링고식 연속 기록 시스템

#### 홈 화면 위젯
```
┌─────────────────────────────────┐
│  🔥 7일 연속 기록!              │
│                                 │
│  월 화 수 목 금 토 일           │
│  ✅ ✅ ✅ ✅ ⬜ ⬜ ⬜         │
│                                 │
│  총 42일 기록  |  이번 주 4/7   │
└─────────────────────────────────┘
```

#### 통계 정보
- **현재 스트릭**: 연속 며칠 기록 중
- **최장 스트릭**: 역대 최고 기록
- **총 기록 일수**: 누적 일수
- **이번 주 기록**: 7일 중 며칠 완료

#### 시각화
- 체크마크 애니메이션
- 스트릭 불꽃 아이콘 🔥
- 7일 달성 시 축하 confetti

### 2. 음식 뱃지 시스템

#### 획득 방식
```
[식사 기록 저장 시]

━━━━━━━━━━━━━━━━━━━
✨ 기록 완료!

🐟 고등어 뱃지 +1
→ 총 3개 획득

🥗 샐러드 뱃지 +1
→ 총 12개 획득

[축하 애니메이션]
━━━━━━━━━━━━━━━━━━━
```

#### 뱃지 카테고리
- 🐟 **단백질 뱃지** (25종): 고등어, 연어, 닭가슴살, 계란...
- 🥗 **채소 뱃지** (30종): 브로콜리, 시금치, 콩나물...
- 🍚 **탄수화물 뱃지** (20종): 현미밥, 고구마, 통밀빵...
- 🍜 **요리 뱃지** (25종): 된장찌개, 제육볶음, 비빔밥...

**총 100종 이상**

#### 업적 시스템
```
[콩나물 100개 달성 시]

━━━━━━━━━━━━━━━━━━━
🎉 업적 달성!

🏆 콩나물 마스터
콩나물 뱃지 100개 획득!

특별 칭호: "해장의 신"
━━━━━━━━━━━━━━━━━━━
```

**업적 예시**:
- 🥚 계란 50개: "계란 덕후"
- 🐟 생선 30개: "오메가3 전문가"
- 🥗 샐러드 20개: "래빗 모드"
- 🍖 고기 금지 7일: "비건 도전자"
- 🍚 현미밥 30개: "건강밥상"

### 3. 소셜 기능 (친구 비교)

#### 친구 프로필 화면
```
┌─────────────────────────────────┐
│  [@김철수]                      │
│                                 │
│  🔥 12일 연속  |  📝 총 58일    │
│                                 │
├─────────────────────────────────┤
│  🏆 획득한 뱃지 (Top 5)         │
│                                 │
│  🐟 고등어 ×15                  │
│  🥚 계란 ×12                    │
│  🥗 샐러드 ×8                   │
│  🍚 현미밥 ×5                   │
│  🥦 브로콜리 ×3                 │
│                                 │
│  [전체 보기 →]                  │
├─────────────────────────────────┤
│  📊 몸무게 비교                 │
│                                 │
│      나      |     철수         │
│   ━━━━━━━━━━━━━━━━━          │
│    68kg     |    72kg          │
│    ↓ 3kg    |    ↓ 5kg         │
│                                 │
│   [상세 그래프 →]               │
└─────────────────────────────────┘
```

#### 친구 리그 시스템
```
📊 이번 주 기록 리그

1위 🥇 @박영희  7/7일 (100%)
2위 🥈 @김철수  6/7일 (86%)
3위 🥉 @나      4/7일 (57%)
4위    @이민수  3/7일 (43%)
```

#### 소셜 액션
- **응원하기**: 친구 프로필에서 응원 보내기
- **선물하기**: 쿠폰을 친구에게 선물
- **비교하기**: 몸무게 그래프 간략 비교

### 4. 보상 시스템 (비용 제로!)

#### A. 디지털 보상

**1. 테마/컬러 잠금 해제**
- 7일 연속 → 🌸 핑크 테마
- 30일 연속 → 🌊 오션 테마
- 100일 누적 → 🌈 레인보우 테마

**2. 칭호 시스템**
- 7일 연속 → 🔥 "불꽃 초보자"
- 30일 연속 → 💪 "습관의 달인"
- 100일 연속 → 👑 "다이어트 마스터"
- 프로필에 칭호 표시

**3. 프로필 꾸미기**
- 뱃지 50종 수집 → 프로필 배경 패턴
- 100일 기록 → 닉네임 반짝이 이펙트

**4. 특별 스티커**
- 샐러드 20회 → 🥗 "건강러" 스티커
- 고등어 10회 → 🐟 "오메가3" 스티커
- 커뮤니티 게시글에 사용 가능

#### B. 현실 연계 보상

**5. 치팅데이 쿠폰**
```
[7일 연속 달성]
→ 🎟️ 치팅데이 쿠폰 1장
→ "오늘은 마음껏 먹어도 괜찮아요!"
→ Streak 유지 (실패해도 안 깨짐)
```

**6. 과자박스 쿠폰**
```
[3일 연속 달성]
→ 🍫 과자박스 쿠폰 1장
→ "작은 간식 하나 드세요!"
→ 기록 시 "쿠폰 사용함" 선택 가능
```

## 구현 우선순위

### Phase 1: MVP (필수) ✅
**목표**: 기본 게임화 시스템 구축

1. **듀오링고식 스트릭 카운터**
   - 홈 화면 위젯
   - 7일 체크박스
   - 총 기록 일수 표시

2. **음식 뱃지 기본 시스템**
   - 50종 뱃지
   - 기록 시 뱃지 획득 알림
   - 프로필에서 뱃지 컬렉션 보기

3. **주간 피드백**
   - 외식/배달 횟수 카운트
   - 경고 메시지

4. **만족도 + 한줄평**
   - 별점 5점
   - 텍스트 입력 (선택)

**완료 기준**:
- 기록 → 뱃지 획득 → 스트릭 증가
- 이번 주 외식 3회 시 경고 표시

### Phase 2: 고급 기능 ⭐
**목표**: 소셜 + 보상 시스템

5. **업적 시스템**
   - 뱃지 누적 업적
   - 칭호 획득

6. **친구 프로필 비교**
   - 친구 스트릭 보기

   - 몸무게 간략 비교

7. **보상 시스템**
   - 테마 잠금 해제
   - 칭호 시스템
   - 쿠폰 (치팅데이, 과자박스)

**완료 기준**:
- 친구 프로필 보기 가능
- 7일 연속 시 치팅데이 쿠폰 획득
- 테마 변경 가능

---

### Phase 3: 미래 확장 🚀
**목표**: 커뮤니티 + 심화 기능

8. **리그 시스템**
   - 친구 랭킹
   - 주간 리그 보상

9. **월간 리포트**
   - 매월 1일 자동 생성
   - 통계 분석

10. **친구 선물하기**
    - 쿠폰 선물
    - 응원 메시지

**완료 기준**:
- 친구와 경쟁/협력 가능
- 월간 리포트 자동 생성

## 심리학 효과

### 이 시스템이 작동하는 이유

1. **Gamification (게임화)**
   - 뱃지 수집 = 컬렉션 욕구
   - 스트릭 = 연속성 욕구
   - 업적 = 성취감

2. **Social Proof (사회적 증거)**
   - 친구 비교 = 경쟁심 + 동기부여
   - 리그 시스템 = 소속감

3. **Tiny Habits**
   - 기록하면 즉각 보상 (뱃지)
   - 작은 성공 = 습관 형성

4. **Self-Compassion**
   - 치팅데이 쿠폰 = 실패 허용
   - "괜찮아요" 메시지

5. **Visual Appeal**
   - 예쁜 뱃지 = 사용 동기
   - 듀오링고 스타일 = 친숙함

6. **Instant Feedback**
   - 즉각적 뱃지 획득
   - 실시간 칭찬 메시지

## 성공 지표
